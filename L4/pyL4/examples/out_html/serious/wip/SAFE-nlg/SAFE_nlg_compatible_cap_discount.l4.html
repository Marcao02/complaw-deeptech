Simple Agreement fo Future Equity (SAFE)

The parameters to the contract are:
  • DEFAULT_ACTION_TIME_LIMIT which is a number of days.
  • PURCHASE_AMOUNT which is a positive USD amount.
  • DISCOUNT_RATE which is a fraction in the range (0,1].
  • VALUATION_CAP which is a positive USD amount.
  • START_INVESTOR_COMMON_STOCKS which is a number of shares.
  • START_INVESTOR_SAFE_PREFERRED_STOCKS which is a number of shares.

  • Scenario: Investment Agreement Signed

    • Investor may Transfer Investment to Company

  • Scenario: Investment Waiting to Convert

    • Company may Initiate an Equity Financing Event
    • Company may Initiate an IPO Liquidity Event
    • Company may Initiate a Change-of-Control Liquidity Event
    • Company may Initiate a Dissolution Event

  • Investor action Transfer Investment to Company

      Define:
        investor_cash := (cast $ (investor_cash - PURCHASE_AMOUNT))
        company_cash := (company_cash + PURCHASE_AMOUNT)
      Go to Transfer Investment to Company

    • Company action Issue SAFE Preferred Stock

        Company must provide:
        • company_capitalization, which is a positive number of shares
        • premoney_valuation, which is a positive USD amount
        Define:
          undiscounted_price_per_share_standard_preferred_stock := (premoney_valuation / company_capitalization)
          cap_price := (VALUATION_CAP / company_capitalization)
          discount_price := (undiscounted_price_per_share_standard_preferred_stock' * DISCOUNT_RATE)
          conversion_price := (min cap_price' discount_price')
          investor_SAFE_Preferred_Stocks := (investor_SAFE_Preferred_Stocks + (PURCHASE_AMOUNT round/ conversion_price'))
        Go to Issue SAFE Preferred Stock

  • Liquidity Events

    • Scenario: Investor to Choose Method of Payment

      • Investor may Choose Payment in Cash
      • Investor may Choose Payment in Common Stock
      • Go to Stock Payment Option Chosen by Default

    • Company action Initiate an IPO Liquidity Event

        Company must provide:
        • company_cash_at_liquidity_event, which is a USD amount
        • liquidity_capitalization, which is a positive number of shares
        • company_valuation, which is a positive USD amount
        Define:
          undiscounted_price_per_share_standard_preferred_stock := (company_valuation / liquidity_capitalization)
          cap_price := (VALUATION_CAP / liquidity_capitalization)
          discount_price := (undiscounted_price_per_share_standard_preferred_stock' * DISCOUNT_RATE)
          conversion_price := (min cap_price' discount_price')
          investor_cash_currently_unconverted := PURCHASE_AMOUNT
          company_cash := company_cash_at_liquidity_event
        Go to Initiate an IPO Liquidity Event

    • Company action Initiate a Change-of-Control Liquidity Event

        Company must provide:
        • company_cash_at_liquidity_event, which is a USD amount
        • liquidity_capitalization, which is a positive number of shares
        • company_valuation, which is a positive USD amount
        • total_investments_of_cashout_investors, which is a positive USD amount
        • reduction_needed_to_qualify_as_usa_tax_free_reorg, which is a USD amount
        Define:
          undiscounted_price_per_share_standard_preferred_stock := (company_valuation / liquidity_capitalization)
          cap_price := (VALUATION_CAP / liquidity_capitalization)
          discount_price := (undiscounted_price_per_share_standard_preferred_stock' * DISCOUNT_RATE)
          conversion_price := (min cap_price' discount_price')
          investor_percent_of_cashout_investor_investments := (check Fraction(0,1] (PURCHASE_AMOUNT / total_investments_of_cashout_investors))
          purchase_amount_reduction : $ := (investor_percent_of_cashout_investor_investments' * reduction_needed_to_qualify_as_usa_tax_free_reorg)
          investor_Common_Stocks := (investor_Common_Stocks + (purchase_amount_reduction round/ conversion_price'))
          investor_cash_currently_unconverted := (check $ (PURCHASE_AMOUNT - purchase_amount_reduction))
          company_cash := (check $ (company_cash_at_liquidity_event - reduction_needed_to_qualify_as_usa_tax_free_reorg))
        Go to Initiate a Change-of-Control Liquidity Event
