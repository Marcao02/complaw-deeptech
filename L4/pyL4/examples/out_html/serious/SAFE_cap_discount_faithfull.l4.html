SAFE
(Simple Agreement for Future Equity)

THIS CERTIFIES THAT in exchange for the payment by CryptoBoy (the Investor) of $100000 (Purchase_Amount) on or about 1 July 2018, Legalese, a Singapore corporation (the Company), hereby issues to the Investor the right to certain shares of the Company’s capital stock, subject to the terms set forth below.

  • Valuation_Cap is $100000000.
  • Discount is (1 - Discount_Rate)%.
  • Discount_Rate is 0.9.

    • The parameters to the contract are:
    • Purchase Amount, which is a positive USD.
    • Discount Rate, which is a fraction in the range (0,1].
    • Discount, which is a fraction in the range [0,1).
    • Valuation Cap, which is a positive USD.
    The contract tracks the following variables:
    • Investor's Common Stock, which is a number of shares with initial value 0.
    • Investor's SAFE Stock, which is a number of shares with initial value 0.
    • Investor's Cash, which is a USD with initial value Purchase Amount.
    • Company Cash Available for Payout, which is a USD.
    • Unconverted Investment, which is a USD with initial value 0.
    • Company's Cash, which is a USD with initial value 0.
    • Conversion Price, which is a positive price in USD/share.
    • Liquidity Cashout, which is a USD with initial value 0.
    • Dissolution Cashout, which is a USD.
